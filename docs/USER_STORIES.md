# USER STORIES
Library Book Management System

US-01: As a librarian, I want to add a new book so that it can be stored in the system.

US-02: As a librarian, I want to prevent duplicate Book IDs to ensure data integrity.

US-03: As a user, I want to borrow an available book from the library.

US-04: As a user, I should not be able to borrow a book that is already borrowed.

US-05: As a user, I want to return a borrowed book to the library.

US-06: As a librarian, I want to generate a library report showing all books and their status.
